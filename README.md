
Express + MongoDB Atlas backend for a helpdesk system.
1. Install dependencies:
   npm install
2. Create `.env` (or use `.env.example`):
   PORT=3000
   - `MONGO_URI=your_mongodb_atlas_url`
   - `JWT_SECRET=your_jwt_secret`
   - `JWT_EXPIRES_IN=1d`
3. Start the server:
   - `npm run dev`
